Primeiro versionamento


1° passo cria a pasta com a aquivo. 2° dentro dessa pasta abri o gir bash e cria o repositorio do projeto com as comandos git init, git add git commit cria a versao. 3° cria o repositorio remoto lá no github e copia o endereço. 4° volta no repositorio e dentro do git bash aplica o git remote add url copiada. 5° apartir daqui toda alteração recebe o git push pra atualizar o remote.
